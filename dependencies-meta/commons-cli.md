## Justification

Apache Commons CLI is used to parse command-line arguments for the `Compiler` main method
(`-i`/`--input` and `-o`/`--output` options, auto-generated help text).

**Why Commons CLI:**
- Apache License 2.0 (permissive)
- Part of the Apache Commons project – a well-maintained, reputable library ecosystem
- Provides automatic help-text generation aligned with project requirements
- Zero transitive runtime dependencies
- One of the most downloaded CLI libraries on Maven Central; stable API for decades
