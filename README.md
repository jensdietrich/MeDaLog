# MeDaLog

A datalog variant with support for modules and annotations, and a compiler that translates
MeDaLog programs into [Souffle](https://souffle-lang.github.io/) Datalog.

## Overview

MeDaLog is a conservative extension of the Souffle dialect of Datalog. It adds two features:

- **Modules** – named, nestable scopes that group rules, facts, and declarations.
- **Annotations** – key/value metadata attached to rules, facts, and modules; inherited from
  enclosing modules unless overridden.

The compiler transforms a MeDaLog program into plain Souffle Datalog by:

1. Adding an `id symbol` slot to every declared predicate.
2. Assigning unique provenance IDs to every fact and rule.
3. Transforming rule heads and bodies so that derived IDs encode their proof structure
   (via Souffle's `cat` function by default).
4. Generating `_module` membership facts for rules/modules inside modules.
5. Generating `_assertedAnnotation` facts and an inheritance rule for annotation propagation.

## Requirements

- Java 17
- Maven 3.8+

## Getting the classpath

```bash
mvn dependency:build-classpath -Dmdep.outputFile=cp.txt
```

The classpath will be written to `cp.txt`. Use it as `<classpath>` below.

## Compiler

### `io.github.bineq.medalog.Compiler`

Compiles a MeDaLog source file into Souffle Datalog.

**Build:**

```bash
mvn package
```

**Run:**

```
java -cp <classpath>:target/medalog-1.0-SNAPSHOT.jar io.github.bineq.medalog.Compiler [options]
```

Options generated by commons-cli:

```
usage: Compiler -i <file> -o <file>
 -i,--input <file>    MeDaLog input file to compile
 -o,--output <file>   Souffle Datalog output file
```

**Example:**

```bash
java -cp <classpath> io.github.bineq.medalog.Compiler -i program.dl -o program.souffle
```

### Static API

The compiler can also be used programmatically:

```java
// String → String
String souffle = Compiler.compile(medaLogSource);

// File → File
Compiler.compile(new File("in.dl"), new File("out.dl"));

// Stream → Stream
Compiler.compile(inputStream, outputStream);

// Reader → Writer
Compiler.compile(reader, writer);

// With custom aggregation strategy
Compiler.compile(source, myStrategy);
```

## Grammar

The ANTLR4 grammar is at `src/main/antlr4/io/github/bineq/medalog/MeDaLog.g4`.

A MeDaLog program extends Souffle Datalog with:

```
// Module declaration
module <name> {
    // Souffle declarations, facts, rules, and nested modules
}

// Annotations (before a rule, fact, or module)
@<key>: <value>
@<key>=<value>
```

Annotation values can be: strings, numbers, booleans, ISO 8601 timestamps, JSON objects, or
JSON arrays.
