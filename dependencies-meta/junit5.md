## Justification

JUnit 5 (Jupiter) is the standard testing framework for Java projects. Used for unit tests,
integration tests, and as the test runner for jqwik property-based tests.

**Why JUnit 5:**
- Eclipse Public License 2.0 (permissive)
- Industry standard; supported by all major IDEs and CI systems
- Modern API with `@Test`, `assertThrows`, `@Nested`, parameterised tests
- Required as a platform for jqwik property-based tests
- Actively maintained by the JUnit team with a large contributor base
