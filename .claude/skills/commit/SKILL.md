---
name: commit
description: Commit staged and unstaged changes with a prompted message, always tagging it as generated by Claude
allowed-tools: "Bash(git:*)"
context: fork
---

1. Run `git status` and `git diff` to summarise what has changed. Show the user a brief summary of the changes.

2. Ask the user for a commit message using AskUserQuestion. The question should be "What is the commit message?" with no predefined options (offer "Other" as the only option so the user types freely).

3. Construct the final commit message by appending ` #generated-by-claude` to whatever the user typed.

4. Stage all modified and untracked files with `git add -A`, then commit using:

```
git commit -m "<user message> #generated-by-claude"
```

   Use a HEREDOC to pass the message to avoid shell interpretation issues.

5. After the commit succeeds, ask the user whether to push using AskUserQuestion with two options:
   - "Push now" — run `git push`
   - "Skip push" — do nothing

6. Report the final outcome (commit hash, whether push happened).
